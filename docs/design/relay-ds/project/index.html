Brand

White-label CRM & call-center platform

A token-driven system, ready to wear any brand.

Every surface, control and status in Relay is painted from semantic tokens — never a hard-coded color. Swap the brand swatch above or flip the theme: the entire system reskins without touching a single component. This is the foundation layer — color, type, space, elevation and motion.

1 brand ramp · 11 stops Full dark-mode parity AA contrast, both themes

Primitive ramps feed semantic tokens. Components only ever reference semantics (--color-primary, --color-fg-2), so a tenant overrides 11 brand stops and is done.
